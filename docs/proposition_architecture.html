🚨
Alerte critique — SHA-1 sur FR/DE/ES/IT
SHA-1 est cryptographiquement cassé depuis 2017. Les mots de passe du plus grand marché (Europe historique) sont stockés avec un algorithme vulnérable. Non-conformité RGPD caractérisée et risque de sécurité immédiat.

3.1 Comparaison des modèles d'architecture

❌ Écarté
Microservices
+ Scalabilité granulaire
− Complexité opérationnelle très élevée
− Équipe DevOps dédiée requise
− Coût infrastructure doublé
❌ Écarté
Serverless
+ Pas de serveur à gérer
− WebSocket difficile en serverless
− Vendor lock-in fort
− Cold start (latence démarrage)
✓ Retenu
Monolithe modulaire
+ Déploiement simple et unique
+ Cohérence des données
+ Équipe unique, outillage unifié
+ Scalable horizontalement
− Scalabilité moins granulaire

⚠
Pourquoi Daphne et pas Gunicorn ?
Gunicorn est le serveur WSGI classique — il ne supporte pas les WebSockets. Dès qu'on intègre Django Channels, on passe obligatoirement en ASGI. Daphne est le serveur ASGI officiel de l'équipe Channels : HTTP + WebSocket simultanément.
